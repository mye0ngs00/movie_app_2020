### 설명
  노마드 코더 리액트 기초 강좌. 첫 번째 리액트 프로젝트.

### 실행방법
`$ git clone https://github.com/mye0ngs00/movie_app_2020`<br/>
`$ cd movie_app_2020`<br/>
`$ npm install && npm start`<br/>


### 스택
- react
- axios
- react-router-dom

### 기능
- 영화 순위 나열
- 영화 세부 정보 확인
